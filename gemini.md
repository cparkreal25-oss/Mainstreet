# Project Constitution: Main Street Research
**Source of Truth: Version 1.0 (Committed April 11, 2026)**

## Project Mission
To modernize the Main Street Research website into a premium, high-end landing page specializing in research and marketing for computer parts.

## Behavioral Rules
- **Deterministic Logic**: "Explore Solutions" triggers an immediate email communication.
- **Data-First**: No logic should be built without a defined data schema.
- **Tool Discipline**: Functions must be small, idempotent, and single-purpose.

## Architectural Invariants
- **Tech Stack**: HTML5, Vanilla CSS, JavaScript.
- **Design System**: Rich aesthetics, HSL color palettes, modern typography (Google Fonts), smooth gradients, and interactive micro-animations.
- **Responsiveness**: Mobile-first design for all pages.

## Data Schemas
### Site Structure
- `index.html`: Hero section (Main Street Research), "Computer Parts Specialists" expertise section, Marketing/Research services, "Explore Solutions" CTA.
- `styles.css`: Dark mode/Premium aesthetics, glassmorphism, high-end typography.
- `script.js`: Smooth scroll, micro-animations, and mailto logic for CTA.
